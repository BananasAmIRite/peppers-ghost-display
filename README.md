# Holographic Pepper's Ghost Display

#### A small widget I custom-developed to spice up my desk (and to learn electronics)!

Inspired by [OpenGhost](https://github.com/xanderchinxyz/OpenGhost)

---

## Features

- ESP32 screen renderer
- Small, low current draw Raspberry PI Zero 2 W as main driver
- UART, SPI interface between ESP32 and Raspberry PI for fast communication
- Raspberry PI AI Camera for gesture recognition
- Custom-developed PCB to connect everything together
- <1.5A@5A total current draw, runnable off USB Hub
